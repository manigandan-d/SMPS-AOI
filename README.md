# SMPS AOI System using Raspberry Pi 5

## Overview
Automatic detection of missing components on SMPS PCB using Raspberry Pi and YOLO.

## Features
- Conveyor-based inspection
- YOLO component detection
- FSM-based control
- PASS/FAIL result UI

## Hardware
- Raspberry Pi 5
- Pi Camera v3 Wide
- DC motor + TB6612FNG
- Conveyor system

## Software
- Python
- Ultralytics YOLO
- NCNN backend
- OpenCV

## How to Run
- python detect_smps.py
